
# React v18
  This is a basic react js v18.
  Node version 18.
  
## Run Locally
Clone the project


```bash
  git clone https://github.com/sombathoudom/react-18-beginners.git
```

Go to the project directory

```bash
  cd react-18-beginners
```

Install Dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
